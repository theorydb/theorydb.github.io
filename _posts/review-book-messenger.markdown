> 

"입력 - 수사 - 출력"의 구조에서 개인적으로 출력 방법은 잘 알고 있으나, 출력되지 못한 수사가 어떤 모양을 갖고 있는지 파악해보고 싶어 본 서를 읽는다. 

* 전달자의 유형  
| 관찰자 | 반복되는 일상을 살펴본다 |  |
| 도전자 | 가치판단 기준이 올바른 덕목인가? |  |
| 파괴자 | 새로운 사유세계 증축 | 니체 |
| 창시자 | 새로운 이정표 제시 |  |
| 수행자 | 꾸준히 삶을 변화시킨다(습관) |  |
| 지도자 | 비전제시 |  |
| 해석자 | 세상 모든 기호의 의미 파악 |  |
| 철학자 | 이면에 숨은 진의 해석 |  |
| 교육자 | 배움 동기부여 |  |
| 동반자 | 가치를 나누며 동행 |  |

* 전달자 예시  
| 소크라테스 | 문답법(산파술): 진의를 스스로 찾을 수 있게 촉매 역할 |
| 메를로 퐁티 | 관념(정신세계) + 경험(몸,오감): 슬픔 |
| 괴테 | 어둠속에서도 한줄기 밝은 빛을 찾아 전달 |
| 톨스토이 | 혼란스러운 세상속에서 내면의 양심과 도덕을 따라가보라고 권유 |
| 니체 | 기존 질서에 굴복하지 않고 아이처럼 순수한 상태에서 창조적으로 자신의 길을 찾아가는 초인(위버멘쉬) 추구 |
| 랑시에르 | 학생 스스로 배울 수 있도록 지도 |
| 헤르만 헤세 | 내면의 고요에 집중하라 |
| 비트겐슈타인 | 언어의 한계는 세상의 한계를 결정한다. 내가 모르는 영역은 보이지 않는다. |
| 돌뢰즈 | 연결 개척자. 낯선 생각들이 충돌하고 교류하고 재구성.  |
| 보르헤스 | 미궁의 진의. 상충하는 해석, 인물들의 다양한 입장, 사회에 미치는 영향 등 다양한 각도에서 |

* 수사  
  * 눈먼각인 / 무딘의미  
  * 비언어적 요소: 경험 등
  * 언어 한계의 틈새
  * 상대방의 언어, 경험 부족 상태: 은유의 다리 필요 
  * 메시지(겉 정보) + 메타 메시지(숨은 진의 및 생각)

* 출력  
  * 손자병법: 듣는이가 원하는 것 / 듣는이가 처한 상황과 이야기 흐름의 맥락
